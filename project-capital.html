Capital Logistics App

Role Full Stack Engineer
Client Capital Logistics
Stack Retool · Node.js · SQL · REST APIs
Type Internal Operations Dashboard

Project Overview

Built a full operations app for Capital Logistics using Retool, giving their team real-time visibility into shipments, driver performance, route efficiency, and API health — all from a single interface.

The backend is powered by Node.js middleware connecting live REST APIs and a PostgreSQL database via optimized SQL queries. Data connections update in real time, enabling dispatch teams to make faster decisions with accurate operational data.
