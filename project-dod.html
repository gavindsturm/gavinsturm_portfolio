03 — LDRedmer · 2025–Present

DoD Secure Infrastructure

Role IT / Security Specialist
Client LDRedmer (DoD-Affiliated)
Stack Windows Server · Active Directory · GPO
Compliance DISA STIGs · NIST · CMMC

Project Overview

Managing and securing a DoD-affiliated intranet server environment for LDRedmer, supporting internal business and operational systems with high availability and stringent security requirements.

Conducted full security hardening using DISA STIGs, implemented least-privilege access controls, configured Group Policy Objects for security enforcement, and set up granular NTFS permissions with RBAC to protect sensitive government data.
