Project Overview

Built a locally hosted AI file search system for LDRedmer's DoD-affiliated environment. The system allows staff to search across their entire document database using natural language — completely offline and air-gapped, with no data leaving the network.

Uses vector embeddings and semantic search to find relevant files based on meaning rather than just keywords. Integrated directly with their existing Windows Server infrastructure and Active Directory permissions, so users only see files they're authorized to access.
