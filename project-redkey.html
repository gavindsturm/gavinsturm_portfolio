Project Overview

REDKEY is a secure file retrieval system built for environments where data sensitivity is paramount. Users can request and retrieve files over encrypted transit, with every request authenticated via rotating API keys that expire after use — eliminating the risk of stolen static credentials.

Files are stored in zero plaintext format — encrypted at rest with AES-256. Decryption only happens locally on the client after authenticated retrieval. Every access event is logged to a tamper-evident audit trail with full RBAC enforcement throughout.
